01 · Sobre mí

Estudiante de Ingeniería en Sistemas con enfoque en desarrollo de software, redes de computadoras, bases de datos y soluciones empresariales. He desarrollado proyectos académicos que integran programación, infraestructura, estructuras de datos y automatización de procesos reales.

Poseo experiencia desarrollando aplicaciones con Python y C, interfaces gráficas con Tkinter, bases de datos MySQL y configuración de servidores Windows Server.

Este portafolio refleja pensamiento lógico, análisis estructurado y dominio técnico para construir sistemas funcionales desde la lógica hasta la implementación.
