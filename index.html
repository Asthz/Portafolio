04 · PROYECTO ESTRELLA: SISTEMA POS "CHINOS CAFÉ"

Este proyecto representa una solución integral de **Punto de Venta (POS)** desarrollada completamente en **Python** con interfaz gráfica **Tkinter**. Está conectado a una base de datos **MySQL** centralizada para la gestión en tiempo real de transacciones, inventarios y facturación.

Su diseño se basó en un entorno comercial real, con el objetivo de **automatizar procesos empresariales críticos** y asegurar una organización estructurada y eficiente de los datos comerciales.

MÓDULOS PRINCIPALES

  • **Gestión de Inventario** (Entradas/Salidas)
  • **Registro de Ventas** (Transacciones)
  • Gestión de **Proveedores**
  • Módulo de **Facturación** e Impresión

STACK TECNOLÓGICO

  • **Lenguaje:** Python 3.x
  • **Frontend/GUI:** Tkinter (nativo de Python)
  • **Base de Datos:** MySQL (conector)
  • **Arquitectura:** Red Local Cliente-Servidor
