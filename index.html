Proyectos del curso

Sistema de alquiler de equipos que incluye algoritmos de búsqueda secuencial y Push Down, diseño de base de datos MySQL, prototipo en Python para gestión de alquileres y diagrama de red LAN en Packet Tracer.

  • Lenguajes: Python
  • Tecnologías: MySQL, Packet Tracer

Examen Parcial 1 – Algoritmos y C

Colección de soluciones en PSeInt y lenguaje C para problemas de conversión de hora, manejo de fechas, cálculo de edad, año bisiesto y salario neto semanal, con simulación de múltiples casos.

  • Lenguajes: C, PSeInt
  • Temas: fechas, ciclos, condicionales
Ver proyecto

Examen Parcial 2 – Sistema POS “CHINOS CAFÉ”

Desarrollo de un sistema de punto de venta para un café familiar. Incluye base de datos CHINOS_CAFE, interfaz POS en Python con Tkinter para gestionar proveedores, inventario, ventas y facturas, conexión a MySQL en máquina virtual y diseño de red con VLAN, enlace entre sedes y firewall en el servidor web.

  • Lenguajes: Python, SQL, PHP (formulario web)
  • Tecnologías: MySQL, Tkinter, Apache, VLAN, UFW
