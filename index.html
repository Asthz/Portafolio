03 · PROYECTOS PRINCIPALES

SISTEMA

TALLER PRACTICO 1

Proyecto que simula el funcionamiento de una empresa de alquiler de equipos. Incluye el análisis de procesos, la aplicación de algoritmos de búsqueda secuencial y manejo de pila (Push Down), el diseño de base de datos relacional en MySQL y un prototipo programado en Python para gestionar los registros de alquileres.

Además, se diseñó la estructura de red LAN en una herramienta de simulación, considerando la comunicación entre los equipos y el servidor de datos.

04 · SISTEMA POS “CHINOS CAFÉ”

Descripción general

CHINOS CAFÉ es un sistema de punto de venta desarrollado en Python con interfaz gráfica (Tkinter) y base de datos MySQL. Está diseñado para gestionar compras, inventario, proveedores y facturación de un negocio real, integrando varios módulos en un solo sistema.

El sistema se conecta a una base de datos en red, permite registrar nuevas ventas, actualizar existencias, consultar información y generar facturas.
