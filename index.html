Detalle: Sistema POS Multicanal

Solución clave diseñada para un cliente minorista con múltiples ubicaciones, requiriendo alta disponibilidad y una experiencia de usuario fluida.

El sistema se construyó utilizando React para el *front-end* (asegurando una UX rápida y responsive en terminales táctiles) y Python (Django Rest Framework) para el *back-end*, lo que permitió un desarrollo ágil y seguro de las APIs.

Uno de los desafíos principales fue la sincronización de inventario. Implementamos un patrón de Event Sourcing con Kafka para asegurar que los cambios de stock fueran consistentes en todas las sucursales en milisegundos.

Además, el módulo de reportes utiliza un almacén de datos separado (Redshift) para evitar la latencia en las operaciones transaccionales. Esto resultó en una mejora del 40% en la velocidad de procesamiento de transacciones diarias.
